Project Overview :- 

This project is about learning and improving basic digital skills through different practical tasks. In task The first task was to create an infrographic on digital awareness. The second one was to create account on few platforms such as GitHub, LinkeIn and Kaggle. The third one was to solve a python coding based challenge on HackerRank and create a google form quize on digital literacy awareness. The fourth task was to write emails and create a social media checklist. The last task was to write a case study on cyber crime and create a preventive checklist for cyber crime.It helped me understand how to use digital tools in a better and more responsible way. Each task was simple but useful and helped me learn something new. In the beginning, I created an infographic on digital literacy using Canva, where I explained important topics like digital tools and safe internet practices in a simple way. Then, I built my digital portfolio by creating profiles on GitHub and LinkedIn, which helped me understand the importance of having an online presence. I also explored coding and collaboration tools. I practiced basic coding on HackerRank and created a quiz using Google Forms, which showed me how easy it is to collect and manage information. After that, I learned about email etiquette by writing professional emails and making a checklist for proper online behavior. In the last task, I worked on a cybercrime case study and learned how to stay safe online. I also organized all my files properly and created a README file to explain my work. Overall, this project was a good learning experience. It helped me improve my digital skills, creativity, and confidence, and showed me how these skills are useful in both studies and real life.
